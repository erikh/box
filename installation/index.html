Getting Box

Download a Release

Installation

Just gunzip the downloaded file and put it in your path:

$ gunzip box.$(uname -s).gz
$ chmod 755 box.$(uname -s)
$ sudo mv box.$(uname -s) /usr/local/bin/box

Alternatively, we have a homebrew tap and debian and redhat packages on the releases page.

Invocation

Use the shell

If you want to try out box quickly, you can use the shell interface, AKA repl (read-eval-print loop):

$ box repl
# or
$ box shell
